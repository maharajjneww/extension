# Quiz Answer Assistant Extension

A browser extension that helps answer MCQ questions using DeepSeek API without switching tabs.

## Features
- Select question text to get instant answers
- Answer appears in top-right corner for 3 seconds
- Click anywhere to dismiss the answer
- No tab switching required - works in the same tab
- Optimized for MCQ questions

## Installation

1. Get your DeepSeek API key from https://platform.deepseek.com/
2. Open Chrome/Edge and go to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top-right)
4. Click "Load unpacked"
5. Select this extension folder
6. Click the extension icon and enter your API key

## Usage

1. During your exam, simply select the question text (with or without options)
2. The answer will appear in the top-right corner
3. Click anywhere to dismiss the answer early
4. Answer auto-disappears after 3 seconds

## Note

You need placeholder icon files (icon16.png, icon48.png, icon128.png) to complete the installation.
You can create simple colored squares as icons or use any image editor.
